환영합니다, 창조자님!
CR8OR는 소환사를 통해 아이템을 소환하고, 아이템을 조합하는 게임입니다. 시작 화면을 보며 간단한 소개를 하겠습니다.

1. 메인화면 
2. 창조자 정보  창조자 닉네임, 레벨, 경험치
3. 메인버튼  각각 홈, 아이템, 제작, 거래소, 캐시샵 화면으로 변경
4. 메뉴버튼  프로필, 백과사전, 문의, 로그아웃



 홈

먼저, 홈 화면에서 첫번째 소환사를 창조해주세요. 소환사를 클릭하면 해당 소환사의 정보를 볼 수 있습니다.

1. 소환사 정보  소환사 레벨, 경험치, 소환 쿨타임(소환 쿨타임마다 소환 아이템 개수 증가 / 기본 쿨타임: 30초), 각성 조건
2. 문양 정보  소환사 현재 장착 문양 정보 (문양: 소환사 강화 아이템)
3. 축복 정보  소환사 현재 축복 정보 (축복: 소환사 일시적 강화 아이템)
4. 소환버튼  축척해놓은 소환 횟수를 모두 소모하고, 아이템 소환
5. 각성버튼  각성 조건에 부합 시, 소환사 각성

 홈에는 소환사 이외에도 토템이 있습니다. 토템은 모든 소환사들의 능력을 향상시켜주는 기물이므로 이후에 창조 재료들을 모아 토템을 창조하여 소환사의 능력을 극대화시켜주세요.



 아이템

 아이템은 기초 조합 재료인 원소와 힘, 토템 창조 재료인 목재, 소환사를 강화시켜주는 축복과 문양, 특정 아이템들을 조합 가능하게 하는 조합서, 기타로 분류됩니다. 아이템 화면에서 아이템들의 정보를 확인하고, 사용할 수 있습니다. 또한, 특정 아이템들은 '오래된 종이 조각'이라는 아이템으로 분해가 가능한데, 이는 이후에 유용하게 사용이 가능한 조합 재료입니다.



 제작

1. 상단 바  아이템 분류 및 구분 설정, 문양 강화 화면으로 변경
2. 아이템 목록  아이템 클릭 시, 정보 확인 및 아이템 제작 가능
3. 제작  아이템 설명 및 제작 재료 확인, 제작 실행
 창조자 레벨에 따라 아이템 등급별 제작 성공 확률이 달라지며, 아이템 등급에 비해 창조자 레벨이 많이 낮은 경우, 제작이 아예 불가능할 수 있습니다. 제작 성공 여부와 별개로 모든 제작 재료들은 소멸됩니다.



 문양 강화

1. 상단 바  제작 화면으로 변경
2. 문양 목록  문양 클릭 시, 정보 확인 및 문양 강화 가능
3. 강화  문양 설명 및 강화 재료 확인, 강화 실행
 문양은 소환사에게 장착 가능한 아이템으로, 소환 쿨타임 감소, 경험치 획득률 증가, 아이템 '창조의 파편'(소환사, 토템의 창조 및 육성에 필요) 획득률 증가, 소환 기준 레벨(소환 기준 레벨이 높아질수록 높은 등급의 아이템 소환 확률이 증가) 증가를 통해 소환사를 강화시켜주는 아이템입니다. 각 부분에 대해서 최대 +5강까지 강화가 가능하며, 모든 부분들이 강화가 완료되면, 다음 단계의 문양 제작 재료로 사용 가능합니다.



 거래소(구매)

1. 검색 바  검색 아이템 조건 설정 및 검색, 판매 화면으로 변경, 보유 골드
2. 아이템 목록  조건에 맞는 아이템 확인 및 구매 가능
3. 페이지 조절  검색 아이템 페이지 조절
 거래소에서 다른 창조자들이 판매 등록한 아이템을 구매할 수 있습니다.



 거래소(판매)

1. 상단 바  보유 아이템 분류 및 판매 목록, 거래소 판매 수수료율, 보유 골드
2. 아이템 목록  아이템 판매 가능(판매 목록 → 판매 취소 / 판매금 수령 가능)
 판매금 수령 시에 상단 바에 있는 거래소 판매 수수료율만큼의 수수료가 지불되므로, 해당 사항 유의하시길 바랍니다.

등급 A(↓) + 2 (A - A(↓)) * 100
확률(%) A(↓) + 1 (1 - A + A(↓)) * 100
